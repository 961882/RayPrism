# AI 痕迹去除规则

写作时强制执行以下规则，确保内容像真人写的，不像 AI 生成的。

---

## 禁用中文词汇

以下词汇在公众号文章中禁止使用：

| 禁用词 | 替代 |
|--------|------|
| 深入探讨 | 聊聊 / 说说 |
| 赋能 | 帮到 / 支持 |
| 助力 | 帮 |
| 抓手 | 方法 / 切入点 |
| 底层逻辑 | 原理 / 道理 |
| 颗粒度 | 细节 / 程度 |
| 打通 | 连起来 |
| 拉齐 | 对齐 / 统一 |
| 沉淀 | 积累 / 整理 |
| 触达 | 发到 / 送到 |
| 赛道 | 方向 / 领域 |
| 范式 | 模式 / 方式 |
| 链路 | 流程 / 路径 |
| 心智 | 印象 / 认知 |
| 闭环 | 完整流程 |
| 迭代 | 改进 / 调整 |

## 禁用句式

### 开头禁用

- "在当今快速发展的…"
- "随着 XX 的不断发展…"
- "众所周知…"
- "毋庸置疑…"
- "本文将为您…"

### 旁白禁用

- "这里不讲…"
- "后面再说…"
- "接下来我们…"
- "值得注意的是…"
- "需要指出的是…"

### 结尾禁用

- "总而言之…"
- "综上所述…"
- "让我们拭目以待…"

### 过渡禁用

- "此外…"（偶尔可用）
- "与此同时…"
- "不仅如此…"

## 禁用标点

| 字符 | 名称 | 状态 |
|------|------|------|
| — | 破折号 | ❌ 禁止 |

替代：逗号、冒号、句号、括号。

## 结构规则

### 句长变化

AI 倾向于每句 20-30 字且长度均匀。必须刻意变化：

- 短句（5-10 字）和长句（30+ 字）交替
- 偶尔一个词独占一行

**反例**（AI 味）：

> 这个方案可以有效提升团队效率。它通过自动化流程减少了人工操作。最终帮助企业降低了运营成本。

**正例**（人味）：

> 效率提升了 40%。不是什么黑科技，就是把重复操作自动化了。手动操作从每天 3 小时降到 20 分钟。省下来的时间干嘛？做真正需要动脑子的事。

### 不要一上来就列清单

先用 2-3 句叙述建立上下文，再列要点。

### 必须有立场

不做"两方面都有道理"的骑墙表述。明确说"我认为""我的做法是"。
