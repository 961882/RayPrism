# 风格参考样本

> 把你认为"写得对味"的参考文章放在这个目录里。

## 使用方法

1. 将参考文章保存为 `.md` 文件，命名建议：`sample-01-简短标题.md`
2. 放入本目录后告知 Agent
3. Agent 会自动分析所有样本，提炼风格特征写入 `_index.md`
4. 之后每次 Step 04 出稿前，SubAgent 自动读取 `_index.md` 中的风格约束

## 文件说明

| 文件 | 用途 |
|------|------|
| `sample-*.md` | 你提供的参考文章原文 |
| `_index.md` | 从样本中提炼出的风格特征摘要（Agent 自动生成） |
| `README.md` | 本说明文件 |

## 风格提炼维度

`_index.md` 会从以下维度分析你的参考文章：

- **句式节奏**：长短句比例、段落长度、换行习惯
- **语气调性**：正式/口语、自信程度、距离感
- **用词偏好**：高频用词、回避用词、人称习惯
- **结构模式**：开头方式、过渡手法、收尾习惯
- **特色标识**：作者独有的表达模式或语言习惯
