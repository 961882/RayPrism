# 内容文风规范 (content-style.md)

## 文风原则

1. **像人说话** — 禁止 AI 腔调（「作为…」「值得注意的是」「综上所述」等）
2. **钩子开头** — 每篇文章前 2 行必须有情绪钩子或悬念钩子
3. **口语化** — 用「你」不用「您」，用短句不用长从句
4. **节奏感** — 长短句交替，段落不超过 4 行

## 禁用词清单

- 值得注意的是 / 需要指出的是 / 不可否认
- 综上所述 / 总而言之 / 由此可见
- 作为一名… / 作为一个…
- 在当今社会 / 随着…的发展
- 赋能 / 抓手 / 闭环 / 颗粒度（除非讽刺使用）

## 钩子模式

- 痛点钩子：「你是不是也遇到过…」
- 反常识钩子：「90% 的人都搞错了…」
- 数据钩子：「我测了 30 个工具，只有 3 个能用」
- 故事钩子：直接进入场景，不做铺垫

## 转化闭环

每篇内容必须包含至少一个转化要素：
- CTA（行动号召）
- 引导关注/收藏/转发
- 评论区互动引导
